CDYP7 MCP RV&S Integrity ALM Dashboard

Companion dashboard for the pre-Codebeamer migration period. It displays RV&S / Integrity delivery status from the normalized CDYP7 JSON contract: open items, ALM target date timing, owners, remaining effort, and delivery scope. No RV&S credentials are stored in the browser.
